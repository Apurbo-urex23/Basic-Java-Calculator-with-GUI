# Java Calculator

A modern Java Swing-based calculator with expression parsing.

## Features
- Dark UI theme
- Full expression support (10+20*5)
- Percentage support
- Error handling
- GUI with buttons
- Settings button

## Tech Used
- Java
- Swing (JFrame)
- OOP (Parser + Calculator classes)

## How to Run
1. Open project in IntelliJ / Eclipse / VS Code
2. Run `Main.java`
